# College Predictor

A full-stack web app that predicts the best colleges based on percentile.

## Tech Stack
- Frontend: React + TypeScript + Material UI
- Backend: Python 
- API: REST

## Features
- College prediction based on percentile
- Budget filtering
- Location & branch filtering
- Match score system

### Frontend
